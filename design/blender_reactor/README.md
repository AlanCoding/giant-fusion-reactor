# Blender Reactor Work

This folder contains a Blender scene script for the linear consensus reactor.

## Outputs

The script is intended to render a handful of views to:

`design/blender_reactor/exports/`

## Model scope

The scene should include:

- the straight plasma channel
- main axial coils
- helical or quasi-helical shaping coils
- trim / correction coils
- end-treatment structures
- multiple camera angles

## Run

From the repo root, run Blender headless with the scene script.

The script is designed to be invoked with Blender and to write PNG exports for several camera views.
