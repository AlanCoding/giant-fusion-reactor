# Orbit-Ratio Constraint

This note shows the direct constraint behind the `B only as required` case.

If the field scales like:

```text
B(a) ∝ 1/a
```

then the thermal gyroradius scales like:

```text
ρ ∝ 1/B ∝ a
```

so the normalized orbit ratio stays flat:

```text
ρ/a = constant
```

That is the point of the minimum-`B` scaling rule: it keeps the orbit-size proxy from getting worse as the machine grows.
